macOS patches

Various bits of my code have ended up in dosdude1's patchers and OpenCore Legacy Patcher over the years since Mojave unsupported my 2010 MacBook.

This only covers parts I helped with; please see the OCLP and Moraea repos to learn about the project as a whole.

dsce

I wrote the dyld shared cache extractor which allows other patches (graphics, wifi, T1) to use libraries from Monterey/Ventura where necessary.

Metal acceleration

I made the original GCN kext, GCN bundle, and Kepler bundle patches for Ventura and most of the bundle interposer for GCN and Broadwell on Sequoia.

All legacy Metal patches also depend on dsce (above).

OpenGL acceleration

I started the project and still occasionally fix bugs; however, the majority of work for Monterey+ is by EduCovas. I also helped with various shared code, Tesla, TeraScale 1, and light mode blurs.
